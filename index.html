AR.js

网页增强现实框架

v 3.2.1 →

性能好

在手机上运行如丝般顺滑。

基于 Web

纯粹的 web 解决方案，因此无需安装。由 Three.js + A-Frame + jsartoolkit5 组成。

开源且标准化

它完全开源免费且适用于任何使用 webgl 和 webrtc 的手机。
